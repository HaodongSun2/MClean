# MClean

Windows 本地软件卸载工具 —— 复古 Win9x/2000 风格 GUI，中英双语切换，绿色便携。

## 特性

- 扫描已安装软件列表（注册表读取）
- 一键调用 Windows 系统卸载器
- 复古 Windows 经典风格界面（Classic UI）
- 中文 / English 双语切换
- 完全免费，纯本地运行，无联网无收集数据

## 系统要求

- Windows 10 / 11（64 位）
- 内存 ≥ 512 MB
- 键盘 + 鼠标
